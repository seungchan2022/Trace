# 코스 편집 UX 개선 실기기 체크리스트 (2026-07-01)

이번 브랜치(`feature/course-edit-ux-panel`)가 만든 기능: 탭 자동 연결(A), 그리기 스트로크=세그먼트(A-2),
지도 위 구간 색상+거리 라벨(B), 실시간 구간 패널(E). 시뮬레이터·유닛테스트는 이미 통과했고,
아래는 실기기에서만 확인 가능한 것들(터치 감, GPS, 제스처 충돌, 실제 라우팅 지연)입니다.

각 시나리오는 "이렇게 해보세요 → 이렇게 되면 정상"입니다. 안 되면 그대로 알려주시면 됩니다.

## 빌드/설치

- [ ] 기기 연결 후 Xcode에서 Run — 정상 설치·실행
- [ ] 첫 실행 시 "신뢰되지 않은 개발자" 알림이 뜨면 설정 > 일반 > VPN 및 기기 관리에서 신뢰

## 권한

- [ ] 첫 실행 시 위치 권한 팝업이 뜬다 → "허용"하면 내 위치 근처로 지도가 열린다
- [ ] (한 번 더 테스트하려면 설정에서 권한을 거부로 바꾼 뒤 재실행) 거부하면 크래시 없이 서울시청 폴백 화면이 뜨고, 알림에서 "설정으로 이동" 버튼이 동작한다

## 시나리오 1 — 탭으로 첫 경로 만들기 (기존 동작, 회귀 확인용)

1. 지도 아무 곳이나 한 번 탭한다 → 초록색 "출발" 핀이 뜬다
2. 다른 곳을 한 번 더 탭한다 → 두 지점 사이에 도보 경로가 그려지고 거리가 표시된다

**정상:** 첫 두 탭은 예전과 동일하게 동작해야 함(대기 핀 → 경로 확정).

## 시나리오 2 — 세 번째 탭부터 자동 연결 (이번에 새로 생긴 동작)

1. 시나리오 1에서 만든 경로가 있는 상태에서, 경로 끝(도착점)과 가까운 곳을 한 번만 탭한다
2. 이번엔 반대로 경로 시작점과 가까운 곳을 한 번만 탭한다

**정상:** 두 경우 모두 탭 한 번만으로 바로 경로가 이어져야 함(예전처럼 두 번 탭할 필요 없음). 더 가까운 쪽(시작 또는 도착)에서 자연스럽게 이어지는지 확인.

## 시나리오 3 — 그리기 모드, 스트로크마다 별도 구간

1. "그리기" 버튼을 눌러 그리기 모드로 전환한다
2. 손가락으로 한 번 그은 뒤 뗀다(스트로크 1) → 잠깐 기다리면 경로가 반영된다
3. 이어서 다른 곳에서 한 번 더 긋는다(스트로크 2)
4. "되돌리기"를 한 번 누른다

**정상:** 스트로크 2개를 그리면 지도 위에 색이 다른 구간 2개가 보여야 함(시나리오 5와 연결). 되돌리기 한 번에 방금 그은 스트로크(마지막 구간) 하나만 없어져야 함 — 전체가 한 번에 사라지면 안 됨.

## 시나리오 4 — 탭과 그리기를 섞어서 이어붙이기

1. 탭으로 구간을 하나 만든다
2. 그리기 모드로 전환해 경로 끝에서 이어서 한 번 긋는다
3. 다시 탭 모드로 전환해 한 번 더 탭한다

**정상:** 모드를 오가도 경로가 끊기지 않고 계속 이어져야 함.

## 시나리오 5 — 지도 위 구간 색상 + 거리 라벨

1. 시나리오 3~4로 구간을 3개 이상 만든다
2. 지도를 손가락으로 팬(이동)하고 두 손가락으로 줌인/줌아웃 해본다

**정상:** 각 구간이 서로 다른 색 선으로 보이고, 구간 중간쯤에 거리 라벨("120m" 등)이 떠 있어야 함. 지도를 움직이거나 확대/축소해도 선이나 라벨이 깜빡이거나 사라지면 안 됨.

## 시나리오 6 — 실시간 구간 패널

1. 구간을 2개 이상 만든 상태에서 지도 우측 상단을 본다 → 총 거리가 작은 칩으로 떠 있다
2. 그 칩을 탭한다 → 목록이 펼쳐지며 구간별 순번·색·거리·누적거리가 보인다
3. 목록에서 아무 구간이나 탭한다
4. 접기 버튼으로 다시 접는다

**정상:** 목록을 탭하면 지도 카메라가 그 구간이 보이도록 이동하고, 해당 구간 선이 더 굵게 강조돼야 함. 패널이 우측 하단의 "내 위치" 버튼을 가리거나, 지도 팬/줌/그리기 조작을 방해하면 안 됨.

## 시나리오 7 — 되돌리기/초기화 후 하이라이트 확인 (이번에 고친 버그)

1. 구간을 3개 만들고 패널에서 아무 구간(예: 2번)이나 탭해 강조시킨다
2. "되돌리기"를 눌러 방금 만든 구간을 지운다
3. 다시 패널을 확인한다

**정상:** 되돌리기 후 남아있는 구간 중 엉뚱한 곳이 강조된 채로 남아있으면 안 됨(강조가 풀리거나, 실제 선택한 구간만 강조돼야 함).

## 제스처/입력 감

- [ ] 그리기 모드에서 손가락으로 긋는 느낌이 버벅이거나 끊기지 않는다
- [ ] 그리기 모드에서 두 손가락 팬(지도 이동)이 자연스럽다
