## 핵심 기능 (손으로 수행)

### 시나리오 1: 탭 전환 — 코스 화면 작업 상태가 유지되는지

**준비:** 앱을 처음 실행한 상태(코스 탭이 기본으로 보임)

**수행:**
1. 지도를 두 번 탭해 경로를 하나 그린다
2. 화면 하단 러닝 탭을 누른다
3. 다시 코스 탭을 누른다

**확인할 것:** 러닝 탭에 갔다 와도 아까 그린 경로와 시트 상태(펼침 정도)가 그대로 남아있는지, 화면이 처음부터 다시 그려지는 느낌 없이 바로 복귀하는지

**결과:** ☐ 통과 ☐ 실패
**메모:**

### 시나리오 2: 러닝 중 탭바가 사라졌다가 돌아오는지

**준비:** 러닝 탭에서 시작 준비된 상태

**수행:**
1. 시작 버튼을 눌러 카운트다운을 지나 트래킹을 시작한다
2. 트래킹 중 화면 하단을 확인한다
3. 종료 버튼을 길게 눌러 러닝을 마친다
4. 요약 화면이 뜬 상태에서 화면 하단을 확인한다
5. 요약 화면을 닫는다

**확인할 것:** 트래킹 시작부터 요약 화면을 닫기 전까지는 하단 탭바가 아예 안 보이는지, 요약을 닫으면 대기 화면으로 돌아오면서 탭바도 다시 나타나는지

**결과:** ☐ 통과 ☐ 실패
**메모:**

### 시나리오 3: 카운트다운 중 탭을 눌러도 괜찮은지

**준비:** 러닝 탭에서 시작 버튼을 누른 직후(카운트다운 진행 중)

**수행:**
1. 카운트다운(3초) 동안 재빨리 코스 탭을 눌러본다
2. 다시 러닝 탭으로 돌아온다

**확인할 것:** 카운트다운이 이상하게 멈추거나 시작이 꼬이지 않는지 — 살짝 어색해도 크래시나 먹통만 아니면 메모만 남기고 계속 진행 (알려진 미세 엣지)

**결과:** ☐ 통과 ☐ 실패
**메모:**

### 시나리오 4: 코스 탭 시트를 올렸다 내렸다 하며 탭바와의 경계 확인

**준비:** 코스 탭, 경로가 하나 그려진 상태

**수행:**
1. 시트를 살짝 눌러 중간 높이로 펼친다
2. 그래버(시트 맨 위 손잡이)를 위로 드래그해 시트를 최대한 끌어올린다
3. 다시 아래로 드래그해 접는다

**확인할 것:**
- 시트 아래쪽과 하단 탭바 사이에 틈이나 겹침이 없는지
- 시트를 접었을 때 손잡이 부분과 탭바가 두 겹으로 겹쳐 보이지 않는지
- 시트를 최대한 끌어올렸을 때, 화면 맨 위(시간이 표시되는 곳)를 시트가 가리지는 않는지 — 그 바로 아래 있던 버튼들(경로 찍기/그리기)은 시트에 거의 다 가려지는 게 정상 동작이다(아주 미세한 틈만 남을 수 있음)
- 시트를 어떤 높이로 두어도 하단 탭바는 항상 보이는지

**결과:** ☐ 통과 ☐ 실패
**메모:**

### 시나리오 5: 플로팅 버튼(되돌리기/앞으로/초기화/현재위치)이 시트를 따라 움직이는지

**준비:** 코스 탭

**수행:**
1. 경로가 없는 상태에서 화면 우측 하단 버튼을 확인한다
2. 지도를 두 번 탭해 경로를 만든다
3. 우측 하단 버튼들을 확인한다
4. 시트를 중간 높이로 올린다
5. 시트를 최대로 올린다

**확인할 것:**
- 경로가 없을 때는 현재 위치 버튼만 보이는지
- 경로를 만들면 되돌리기/앞으로/초기화 버튼이 함께 나타나는지
- 시트를 올릴수록 버튼들이 시트 위쪽으로 따라 올라가면서 점점 옅어지는지
- 시트를 최대로 올리면 버튼이 사라지고, 그 자리를 눌러도 아무 반응이 없는지

**결과:** ☐ 통과 ☐ 실패
**메모:**

### 시나리오 6: 가로로 기기를 돌렸을 때 하단 탭바

**준비:** 아무 탭이나 (코스 탭 권장)

**수행:**
1. 기기를 가로 방향으로 돌린다
2. 하단 탭바를 확인한다
3. 다시 세로로 돌린다

**확인할 것:** 가로 모드에서 탭바가 세로일 때보다 화면을 과하게 많이 차지하거나 무겁게 처져 보이지 않는지, 세로로 돌아왔을 때 탭바가 원래 크기로 정상적으로 보이는지

**결과:** ☐ 통과 ☐ 실패

## 다크/라이트 모드

### 시나리오 7: 두 테마 모두에서 탭바·시트 경계가 자연스러운지

**준비:** 코스 탭

**수행:**
1. 설정에서 라이트 모드로 전환한다
2. 시트를 이런저런 높이로 올렸다 내렸다 하며 하단 탭바와의 경계를 확인한다
3. 다크 모드로 전환해 같은 것을 확인한다

**확인할 것:** 두 모드 모두에서 시트와 탭바 사이 경계선이 자연스럽고, 색이 서로 부딪히거나 이상하게 보이지 않는지

**결과:** ☐ 통과 ☐ 실패
